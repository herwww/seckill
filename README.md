# Java高并发秒杀API
## 环境

- jdk 1.8

- tomcat 9.0

## 已完成

- 慕课网视频中程序

## 使用方法

1. ````shell
   git clone https://github.com/saaak/seckill.git
   ````

2. ````
   在mysql中创建数据库，并导入 \src\main\sql 目录下的sql文件
   ````

3. ````
   安装并启动 redis
   ````

4. ````
   修改 resources 目录下的 Config.properties 文件中的配置
   ````

5. ````
   启动 tomcat
   ````

6. ````
   访问 http://localhost:8080/seckill/list 
   ````
